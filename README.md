## Run development server

Run the following commands in the terminal

```bash
  cd minestyle
  # Or the name you gave the project when cloning the repository

  # Install dependencies
  npm install
  # or
  pnpm install # recommended
  # or
  yarn install

  # Run dev server
  npm dev
  # or
  pnpm dev
  # or
  yarn dev
```
